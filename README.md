# Simplicity

Simplicity gives you tools to write more concise ActionController and helper methods.

## Dependencies

Requires Ruby 1.9.x

## Installation

Add this line to your application's Gemfile:

    gem 'simplicity'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simplicity
